= regis-cli Analysis Report
*Final report envelope produced by regis-cli, containing request metadata and analyzer results.*
== Properties
* [[properties_version]]**`version`** *(string, required)*: Version of regis-cli that generated this report.
* [[properties_metadata]]**`metadata`** *(object)*: Arbitrary user-provided metadata. Can contain additional properties.
* [[properties_links]]**`links`** *(array)*: Custom templated links.
** [[properties_links_items]]**Items** *(object)*
*** [[properties_links_items_properties_label]]**`label`** *(string, required)*
*** [[properties_links_items_properties_url]]**`url`** *(string, required)*
* [[properties_request]]**`request`** *(object, required)*: Metadata describing the analysis request. Cannot contain additional properties.
** [[properties_request_properties_url]]**`url`** *(string, required)*: Original URL or image reference provided by the user.
** [[properties_request_properties_registry]]**`registry`** *(string, required)*: Resolved registry hostname (e.g. registry-1.docker.io).
** [[properties_request_properties_repository]]**`repository`** *(string, required)*: Full repository path (e.g. library/nginx).
** [[properties_request_properties_tag]]**`tag`** *(string, required)*: Image tag that was analyzed.
** [[properties_request_properties_digest]]**`digest`** *(string)*: Resolved image manifest digest (e.g. sha256-xxx), if available.
** [[properties_request_properties_analyzers]]**`analyzers`** *(array, required)*: List of analyzer names that were executed.
*** [[properties_request_properties_analyzers_items]]**Items** *(string)*
** [[properties_request_properties_timestamp]]**`timestamp`** *(string, format: date-time, required)*: ISO 8601 UTC timestamp of the analysis.
** [[properties_request_properties_metadata]]**`metadata`** *(object)*: Arbitrary user-provided metadata. Can contain additional properties.
* [[properties_results]]**`results`** *(object, required)*: Analyzer results keyed by analyzer name. Can contain additional properties.
** [[properties_results_additionalProperties]]**Additional properties** *(object)*
* [[properties_playbooks]]**`playbooks`** *(array)*: List of evaluated playbook results.
** [[properties_playbooks_items]]**Items**: Refer to *[playbook_result.schema.json](:///playbook_result.schema.json#)*.
* [[properties_playbook]]**`playbook`**: Primary playbook result (shorthand for playbooks[0]). Refer to *[playbook_result.schema.json](:///playbook_result.schema.json#)*.
