= Playbook Definition
*Schema for regis-cli playbook definition files (YAML or JSON).*
== Properties
* [[properties_name]]**`name`** *(string, required)*: Display name of the playbook.
* [[properties_description]]**`description`** *(string)*: Human-readable description of what this playbook evaluates.
* [[properties_slug]]**`slug`** *(string)*: Short identifier used for HTML report filename generation.
* [[properties_links]]**`links`** *(array)*: Optional custom links to display as actions for this playbook.
** [[properties_links_items]]**Items** *(object)*
*** [[properties_links_items_properties_label]]**`label`** *(string, required)*
*** [[properties_links_items_properties_url]]**`url`** *(string, required)*: URL template which can use {metadata[key]} placeholders or Jinja2 syntax.
*** [[properties_links_items_properties_condition]]**`condition`**: Optional JsonLogic expression to determine if the link should be displayed.
* [[properties_pages]]**`pages`** *(array)*: List of playbook pages. Each page groups sections. Length must be at least 1.
** [[properties_pages_items]]**Items**: Refer to *<<%24defs_page,#/$defs/page>>*.
* [[properties_sections]]**`sections`** *(array)*: List of playbook sections. Each section groups scorecards, levels, and display preferences. Length must be at least 1.
** [[properties_sections_items]]**Items**: Refer to *<<%24defs_section,#/$defs/section>>*.
* [[properties_sidebar]]**`sidebar`** *(object)*: Optional sidebar navigation configuration.
** [[properties_sidebar_properties_sections]]**`sections`** *(array)*
*** [[properties_sidebar_properties_sections_items]]**Items** *(object)*
**** [[properties_sidebar_properties_sections_items_properties_title]]**`title`** *(string)*
**** [[properties_sidebar_properties_sections_items_properties_links]]**`links`** *(array, required)*
***** [[properties_sidebar_properties_sections_items_properties_links_items]]**Items** *(object)*
****** [[properties_sidebar_properties_sections_items_properties_links_items_properties_label]]**`label`** *(string, required)*
****** [[properties_sidebar_properties_sections_items_properties_links_items_properties_url]]**`url`** *(string, required)*
****** [[properties_sidebar_properties_sections_items_properties_links_items_properties_icon]]**`icon`** *(string)*
** [[properties_sidebar_properties_links]]**`links`** *(array)*
*** [[properties_sidebar_properties_links_items]]**Items** *(object)*
**** [[properties_sidebar_properties_links_items_properties_label]]**`label`** *(string, required)*
**** [[properties_sidebar_properties_links_items_properties_url]]**`url`** *(string, required)*
**** [[properties_sidebar_properties_links_items_properties_icon]]**`icon`** *(string)*
* [[properties_integrations]]**`integrations`** *(object)*: Optional third-party platform integrations (e.g. GitLab, GitHub).
** [[properties_integrations_properties_gitlab]]**`gitlab`** *(object)*
*** [[properties_integrations_properties_gitlab_properties_labels]]**`labels`** *(array)*: Custom scoped labels applied to GitLab Merge Requests based on conditions.
**** [[properties_integrations_properties_gitlab_properties_labels_items]]**Items** *(object)*
***** [[properties_integrations_properties_gitlab_properties_labels_items_properties_name]]**`name`** *(string, required)*: Label name, e.g. 'regis::fail' or 'security::critical'.
***** [[properties_integrations_properties_gitlab_properties_labels_items_properties_condition]]**`condition`** *(object, required)*: JsonLogic expression evaluated against the analysis report.
*** [[properties_integrations_properties_gitlab_properties_checklist]]**`checklist`** *(array)*: Checklist items added as checkboxes to the Merge Request description.
**** [[properties_integrations_properties_gitlab_properties_checklist_items]]**Items** *(object)*: Cannot contain additional properties.
***** [[properties_integrations_properties_gitlab_properties_checklist_items_properties_label]]**`label`** *(string, required)*: Text of the checkbox item.
***** [[properties_integrations_properties_gitlab_properties_checklist_items_properties_show_if]]**`show_if`** *(object)*: Optional JsonLogic expression. If provided, the item is only included when the expression evaluates to truthy.
***** [[properties_integrations_properties_gitlab_properties_checklist_items_properties_check_if]]**`check_if`** *(object)*: Optional JsonLogic expression. If provided and evaluates to truthy, the checkbox renders pre-checked (- [x]). Otherwise it renders unchecked (- [ ]).
*** [[properties_integrations_properties_gitlab_properties_templates]]**`templates`** *(array)*: URLs to Cookiecutter templates that will be rendered and added to the Merge Request branch.
**** [[properties_integrations_properties_gitlab_properties_templates_items]]**Items** *(object)*: Cannot contain additional properties.
***** [[properties_integrations_properties_gitlab_properties_templates_items_properties_url]]**`url`** *(string, required)*: Cookiecutter template URL or path.
***** [[properties_integrations_properties_gitlab_properties_templates_items_properties_directory]]**`directory`** *(string)*: Optional subdirectory within the repository containing the template.
***** [[properties_integrations_properties_gitlab_properties_templates_items_properties_condition]]**`condition`**: JSON Logic expression to conditionally render the template. Refer to *[jsonlogic.schema.json](:///jsonlogic.schema.json#)*.
== Definitions
* [[%24defs_page]]**`page`** *(object)*: A playbook page containing sections. Cannot contain additional properties.
** [[%24defs_page_properties_title]]**`title`** *(string, required)*: Display name of the page.
** [[%24defs_page_properties_slug]]**`slug`** *(string)*: Short identifier used for HTML report filename generation. If not provided, it falls back to the playbook slug.
** [[%24defs_page_properties_sections]]**`sections`** *(array, required)*: List of playbook sections. Length must be at least 1.
*** [[%24defs_page_properties_sections_items]]**Items**: Refer to *<<%24defs_section,#/$defs/section>>*.
* [[%24defs_section]]**`section`** *(object)*: A playbook section containing scorecards, optional levels, and display preferences. Cannot contain additional properties.
** [[%24defs_section_properties_name]]**`name`** *(string)*: Display name of the section.
** [[%24defs_section_properties_hint]]**`hint`** *(string)*: Optional informative text displayed below the section name.
** [[%24defs_section_properties_display]]**`display`**: Refer to *<<%24defs_display,#/$defs/display>>*.
** [[%24defs_section_properties_levels]]**`levels`** *(array)*: Priority/severity levels used to group scorecards. Built-in fallback order exists for bronze, silver, gold.
*** [[%24defs_section_properties_levels_items]]**Items**: Refer to *<<%24defs_level,#/$defs/level>>*.
** [[%24defs_section_properties_scorecards]]**`scorecards`** *(array)*: Evaluation scorecards with JsonLogic conditions.
*** [[%24defs_section_properties_scorecards_items]]**Items**: Refer to *<<%24defs_scorecard,#/$defs/scorecard>>*.
** [[%24defs_section_properties_widgets]]**`widgets`** *(array)*: KPI and Template widgets displayed in the section.
*** [[%24defs_section_properties_widgets_items]]**Items**: Refer to *<<%24defs_widget,#/$defs/widget>>*.
** [[%24defs_section_properties_condition]]**`condition`** *(object)*: Optional jsonLogic expression to conditionally display this section. If it evaluates to falsy, the section is hidden.
* [[%24defs_display]]**`display`** *(object)*: Rendering preferences for the section. Cannot contain additional properties.
** [[%24defs_display_properties_analyzers]]**`analyzers`** *(array)*: List of analyzer names whose output should be embedded in this section.
*** [[%24defs_display_properties_analyzers_items]]**Items** *(string)*
** [[%24defs_display_properties_widgets]]**`widgets`** *(array)*: KPI widgets displayed in the section header.
*** [[%24defs_display_properties_widgets_items]]**Items**: Refer to *<<%24defs_widget,#/$defs/widget>>*.
* [[%24defs_widget]]**`widget`** *(object)*: A key-value widget displaying a metric from the analysis report, or a custom HTML template widget. Cannot contain additional properties.
** [[%24defs_widget_properties_label]]**`label`** *(string)*: Display label for the widget.
** [[%24defs_widget_properties_value]]**`value`** *(string)*: Dot-separated path into the report data, e.g. 'results.trivy.critical_count'.
** [[%24defs_widget_properties_url]]**`url`** *(string)*: Optional URL for the widget. Supports Jinja2 templates.
** [[%24defs_widget_properties_icon]]**`icon`** *(string)*: Emoji or icon displayed alongside the widget.
** [[%24defs_widget_properties_template]]**`template`** *(string)*: Path to a Jinja2 HTML template within the theme, e.g. analyzers/trivy/table.html.
** [[%24defs_widget_properties_options]]**`options`** *(object)*: Arbitrary options passed directly to the Jinja2 template.
*** [[%24defs_widget_properties_options_properties_title]]**`title`** *(string)*: Optional title for the widget. If provided, the widget will be displayed with a header.
*** [[%24defs_widget_properties_options_properties_collapsed]]**`collapsed`** *(boolean)*: If true, the widget will be collapsible and closed by default. Default: `false`.
*** [[%24defs_widget_properties_options_properties_align]]**`align`** *(string)*: Text alignment for the widget (left, center, or right). Must be one of: "left", "center", or "right". Default: `"left"`.
*** [[%24defs_widget_properties_options_properties_subvalue]]**`subvalue`** *(string)*: Additional text/value to display below the main value. Follows identical resolution logic as value.
*** [[%24defs_widget_properties_options_properties_class]]**`class`** *(string)*: Additional CSS class(es) to apply to the widget container. Default: `""`.
** [[%24defs_widget_properties_condition]]**`condition`** *(object)*: Optional jsonLogic expression to conditionally display this widget. If it evaluates to falsy, the widget is hidden.
* [[%24defs_level]]**`level`** *(object)*: A priority/severity level used to group and summarise scorecards. Cannot contain additional properties.
** [[%24defs_level_properties_name]]**`name`** *(string, required)*: Level identifier referenced by scorecards.
** [[%24defs_level_properties_label]]**`label`** *(string)*: Human-readable display label.
** [[%24defs_level_properties_order]]**`order`** *(integer)*: Sort order (lower value = higher priority). Built-in defaults: bronze=1, silver=2, gold=3.
* [[%24defs_scorecard]]**`scorecard`** *(object)*: An evaluation scorecard with a JsonLogic condition. Cannot contain additional properties.
** [[%24defs_scorecard_properties_name]]**`name`** *(string, required)*: Unique identifier for the scorecard.
** [[%24defs_scorecard_properties_title]]**`title`** *(string)*: Human-readable title. Defaults to name if omitted.
** [[%24defs_scorecard_properties_level]]**`level`** *(string)*: Level this scorecard belongs to. Must match a level name defined in the section.
** [[%24defs_scorecard_properties_tags]]**`tags`** *(array)*: Arbitrary tags for filtering or grouping.
*** [[%24defs_scorecard_properties_tags_items]]**Items** *(string)*
** [[%24defs_scorecard_properties_condition]]**`condition`**: JsonLogic expression evaluated against the flattened analysis report. Variables use dot-paths, e.g. {"var": "results.trivy.critical_count"}. Supported operators: ==, !=, >, >=, <, <=, in, !, and, or.
