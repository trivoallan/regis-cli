= regis-cli

`regis-cli` is a powerful tool to analyze Docker image registries, evaluate security playbooks, and produce rich HTML/JSON reports.

For a detailed look at how the tool works and its internal structure, see the xref:overview.adoc[Architectural Overview] and xref:playbooks.adoc[Understand Playbooks].
